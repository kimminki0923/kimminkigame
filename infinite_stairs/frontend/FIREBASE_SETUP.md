# 🔥 구글 로그인 설정 가이드 (3분 컷)

구글 로그인을 작동시키려면, 구글에게 "이 웹사이트에서 로그인을 쓸 거야"라고 신고하고 **키(Key)**를 받아와야 합니다.
아래 순서대로 따라하시면 됩니다.

## 1. Firebase 프로젝트 만들기
1. [Firebase 콘솔(클릭)](https://console.firebase.google.com/)에 접속해서 구글 아이디로 로그인하세요.
2. **[프로젝트 만들기]** 버튼을 누르세요.
3. 프로젝트 이름에 `Infinite Stairs`라고 적고 **[계속]** -> **[계속]** -> **[프로젝트 만들기]**를 누르세요. (애널리틱스는 꺼도 됩니다)

## 2. 웹 앱 추가하기
1. 프로젝트가 만들어지면 화면 중앙에 있는 **웹 아이콘 ( </> )**을 클릭하세요.
2. 앱 닉네임을 대충 적고 **[앱 등록]**을 누르세요.
3. 그러면 **`const firebaseConfig = { ... }`**라고 적힌 코드가 나옵니다.
4. **그 코드 뭉치를 통째로 복사하세요.**

## 3. 내 코드에 붙여넣기
1. VSCode에서 `frontend/auth.js` 파일을 여세요.
2. 맨 윗부분에 제가 `// 여기에 붙여넣으세요!`라고 표시해둔 곳이 있습니다.
3. 기존의 `const firebaseConfig = { ... };`를 지우고, 방금 복사한 코드를 **그대로 붙여넣기** 하세요.

---

## 4. 마지막 중요 설정 (로그인 허용)
다시 Firebase 콘솔 사이트로 돌아가서:
1. 왼쪽 메뉴에서 **[빌드]** -> **[Authentication]**을 클릭하세요.
2. **[시작하기]** 버튼을 누르세요.
3. **[로그인 방법]** 탭에서 **[Google]**을 찾아서 클릭하세요.
4. 오른쪽 위에 **[사용 설정]** 스위치를 켜고, **지원 이메일**을 선택한 뒤 **[저장]**을 누르세요.
